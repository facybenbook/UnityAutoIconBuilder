# UnityAutoIconBuilder
Unityでビルド時に自動でアイコンを生成し設定してくれるものです。

[UnityIconCustomizer](https://github.com/makoto-unity/UnityIconCustomizer)をベースに、

* PC向けに出力サイズを変更 
* ビルド時に自動生成する

といった改造をしています。


# 使用法
[こちら]()からunitypackageを取得し、自分のプロジェクトに展開してください。

あとは自動でビルド時にテクスチャが設定されます！
